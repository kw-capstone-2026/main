# 프로젝트 진행 리포트 (2026-05-10)

## 1. 금일 작업 요약

- **데이터 확보:** 서울시 상권분석서비스 ~(2019~2024+) 데이터(점포, 매출, 유동인구) 확보 및 검증 완료.
- **공간 데이터 정비:** 상권영역(Polygon) Shapefile 확보 및 상권 코드(`TRDAR_CD`) 매핑 확인.
- **파이프라인 구축:** GPU 가속(cuDF)을 지원하는 데이터 엔지니어링 파이프라인(`agenttest/data_pipeline_v1.py`) 구축.
- **통합 데이터셋 생성:** 512,626행 규모의 분석용 통합 데이터셋(`data/processed_commercial_data.parquet`) 생성 완료.
  - 파생 변수 생성: 유효 수요 전환율, 점포당 평균 매출, YoY 폐업 증감률 등.
  - 타겟 라벨링: 매출 하락 및 폐업률 기반 '과포화 위험 여부(0/1)' 정의.

## 2. 팀원 협업 요청 내역 (DS1, DS5)

팀원들에게 프로젝트 규격 통일 및 고도화 작업을 요청함.

### 공통 규격 약속

- **포맷:** `.parquet`
- **조인 키:** `상권_코드`, `기준_년분기_코드`, `서비스_업종_코드`
- **배포 자료:** `서울시_상권영역_맵데이터_통합본.zip` 공유 (공간 매핑 기준점)

### 팀원별 추가 요청 사항

- **(DS1 상가정보):** 상권 코드 매핑, 업종 밀집도(경쟁 강도), 프랜차이즈 비율, 평균 생존 주기 분석.
- **(DS5 격자 인구):** 격자 인구의 상권 단위 합산, 주야간 인구 비율, 인구 변화 추세(Slope) 분석.
